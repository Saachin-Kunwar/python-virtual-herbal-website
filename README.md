# 🚀 Features

## 🌿 Medicinal Plant Information System
Explore detailed medicinal plant data from multiple herbal traditions:

- 🌱 Plant Name  
- 🧬 Scientific Name  
- 💊 Medicinal Benefits  
- 🌍 Herbal Traditions  

Supports:
- Ayurveda  
- Unani  
- Siddha  
- Homeopathy  

---

## 🔍 Smart Plant Search System
Users can search plants instantly by:
- Plant Name  
- Medicinal Benefits  

Features:
- Real-time filtering  
- Case-insensitive search  
- Fast plant lookup  

---

## 🤖 Plant Recommendation System
Interactive recommendation engine based on health concerns.

Supports:
- Stress  
- Immunity  
- Skin Care  
- Digestion  
- Pain Relief  
- Respiratory Problems  
- Anxiety  
- Fever  
- Memory  

Example:
- Stress → Ashwagandha, Brahmi  
- Immunity → Tulsi, Neem  

---

## 📊 Herbal Statistics Dashboard
Displays:
- Total Herbal Traditions  
- Total Medicinal Plants  
- Countries Covered  

---

## 📈 Interactive Data Visualization
- Immunity comparison  
- Antibacterial analysis  
- Stress relief analysis  

Built using Plotly charts.

---

## 🌍 Herbal Garden Map
Shows herbal locations using latitude & longitude.

---

## 🧠 Herbal Quiz System
- Multiple choice questions  
- Score calculation  
- Instant results  
- Celebration animation 🎈  

---

## ⬇️ CSV Download
Download medicinal plant data as CSV file for:
- Study  
- Research  
- Analysis  

---

## 🎨 Modern UI
- Dark theme  
- Green herbal design  
- Custom CSS cards  
- Responsive layout
